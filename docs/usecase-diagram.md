# Use Case Diagram – Online Exam Proctoring System

## Actors:
1. Student
2. Admin

## Use Cases:

### Student:
- Login
- Take Exam
- Submit Exam
- View Result

### Admin:
- Login
- Add Questions
- Manage Students
- Monitor Exam (Proctoring)

## Description:

The Online Exam Proctoring System allows students to log in and take exams online.
After completing the exam, they can submit it and view their results.

The Admin manages the system by adding questions, managing student data,
and monitoring exams to prevent cheating.

## Relationships:

- Student → interacts with → Exam System
- Admin → controls → System Operations
